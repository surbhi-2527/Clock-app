# ⏰ React Clock, Timer & Stopwatch

A simple and interactive **React application** that includes a **live digital clock, countdown timer, and stopwatch** — similar to the features of a digital alarm clock. The app demonstrates the use of **React Hooks (`useState`, `useEffect`)** to manage time-based updates and user interactions.

## ✨ Features

* **Live Clock** – Displays the current system time and updates every second.
* **Countdown Timer** – Allows users to enter time in seconds and start a countdown.
* **Stopwatch** – Measures elapsed time with precision in **hours : minutes : seconds : milliseconds**.
* **Start, Stop, and Reset controls** for both the timer and stopwatch.
* **Clean and responsive UI** for easy interaction.

## 🛠️ Technologies Used

* **React.js**
* **JavaScript (ES6+)**
* **HTML5**
* **CSS3**
* **React Hooks** (`useState`, `useEffect`)

## ⚙️ How It Works

* The **live clock** updates every second using `setInterval`.
* The **countdown timer** decreases the entered time every second until it reaches zero and notifies the user.
* The **stopwatch** counts elapsed time in milliseconds and formats it into a readable time format.

## 🚀 How to Run the Project

1. Clone the repository
2. Install dependencies
3. Start the development server

Example:

```
npm install
npm start
```

The app will run locally on:

```
http://localhost:3000
```

## 📌 Purpose

This project was created to practice **React fundamentals**, especially working with **state management and side effects** for time-based applications.
